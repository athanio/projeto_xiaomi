### Opção 2: Manual (Se automático demorar)
1. Acesse https://app.netlify.com
2. Vá para seu site/projeto
3. Clique em "Deploys"
4. Clique no menu "..." do deploy mais recente
5. Selecione "Retry deploy" ou "Deploy from GitHub"

```bash

npm install -g netlify-cli

# Faça login
netlify login

# Redeploy
netlify deploy --prod
```
